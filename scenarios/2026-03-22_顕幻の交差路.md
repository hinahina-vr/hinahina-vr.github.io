# 顕幻の交差路 ─ 創作者は自分の世界から出られるのか ─

- **ジャンル**: 顕幻迷宮ノベル
- **日付**: 2026-03-22（日）
- **テーマ**: 創作者が自分で作った A.D.M.S. の中に落ち、分岐と再合流を身体で辿る
- **実装ファイル**: `scenarios/2026-03-22_顕幻の交差路.json`

## 実装方針

- 旧「100部屋を総当たりする迷宮」案は採用しない。
- 現行実装は、四層構造の顕界ルートに対して10本の幻界サブが横から食い込む形にする。
- 重要なのは枝の本数よりも、どこで再合流し、どこで戻れなくなるかが見えること。
- dream-select から見えるノード名と、シナリオ JSON の `label` 名を一致させる。

## サブシナリオ接続ルール

- すべての `2026-03-22_*` サブシナリオは `standalone_start` と `entry_from_main` の二重入口を持つ。
- dream-select から直接開いた場合は `standalone_start` から始める。
- 本編 `loadScenario` で落ちる場合は必ず `entry_from_main` に入り、重複した導入文を出さない。
- 本編からの遷移では URL を更新しない。フラグ、backlog、サイトモードは保持する。

## 期待する見え方

- dream-select 上では、`living_foreshadow` / `kitchen_fridge` / `roof_dejiko` / `html_world` / `diary_world` が本線の途中に差し込まれた横枝として見える。
- `living_explore`、`kitchen_oji`、`room_pc`、`resident_hub` が再合流点として見える。
- 住人部屋の6本と、郵便受け/コード深層/無音配信の3本が、同一夜の別圧として第四層の手前と先に並ぶ。
