# 百夢回廊 ─ 幻界サブシナリオ設計書（実装同期版）

各サブシナリオは独立して読めるが、本編から落ちたときは本編の温度を維持したまま始まる。  
旧草案にあった「みとら導入」は採用しない。現行実装は `standalone_start` / `entry_from_main` の二重入口で管理する。

## 共通ルール

- `standalone_start`: dream-select や `?scenario=<sub>` の直リンク用。単独再生であることだけを短く示す。
- `entry_from_main`: 本編 `2026-03-21_百夢回廊` の `GEN_*` 分岐から落ちた直後の入口。ここでは説明を挟まず、最初の固有イメージから始める。
- 本編からの接続では、BGM、背景、flags、backlog を保持したまま遷移する。
- サブシナリオ内で顕界へ戻る再分岐は持たせない。今回は terminal branch として扱う。

## 書き方の指針

- `entry_from_main` では「これは何々世界線の記録」と説明しない。選択の結果がそのまま景色に変わったように始める。
- 単独再生時の説明は `standalone_start` に閉じ込め、本文の固有イメージを汚さない。
- 顕界本編で積み上げた問いを、そのサブのゲストキャラ一人に集中させる。
- 本編の文章をコピーしない。同じ夜から落ちても、最初の一段目から別の視界を見せる。
