# 百夢回廊 ── 幻界サブシナリオ設計書（10本）

各サブシナリオは独立したプレイアブルシナリオ（80〜150行/2000文字以上）。
メインシナリオからシームレスに接続。dream-selectからも個別に遊べる。

**共通フォーマット**:
1. `bg: abyss` → みとら導入
2. `bg` 切替 → ゲストキャラとワディー（幻影）の対話劇
3. `bg: abyss` → みとらの哲学的考察
4. 区切り → みとら「幻界の記録は、ここまでだ」→ ナレーション締め
5. `ending` で終了
