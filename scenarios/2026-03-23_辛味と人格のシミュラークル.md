## コンセプト

### 中心構造: シミュラークルの五重奏

今回のシナリオは「偽の入力、本物の出力」という構造を五つの層で変奏する。

ジャン・ボードリヤールが名づけた「シミュラークル」──実体のない記号が、実体以上のリアリティを持ってしまう現象──を、プレイヤーは身体感覚として辿る。辛味という最も原始的なシミュラークル（火事が起きていないのに火事だと感じる）から出発し、より複雑なシミュラークル（AIが演じる人格、他者が記録する人生）へと階段を上っていく。

1. **辛味の層**（第一層）: カプサイシンがTRPV1受容体に結合し、42度以上の熱を検知したと脳に偽の信号を送る。偽の火事。脳は偽の火事に対して本物のエンドルフィンを放出する。プレイヤーは蒙古タンメン中本でゴモミ（五目味噌タンメン・辛さ8）を注文し、この構造を舌で体験する。

2. **人格の層**（第二層）: IPホルダーがキャラクターの設定──口調、一人称、好き嫌い、NG項目──をAIに読ませると、AIはそのキャラクターになる。偽の人格が本物の言葉を出力する。プレイヤーは町田の夜を歩きながら、ゲームキャラクターの自律性とカドカワ予言の射程に思い至る。

3. **キャラクターの層**（第三層）: 大奥AIの内部に入り、32人のキャラクターが住む部屋を巡る。各キャラクターは自分がパラメータの産物であることを知っているのか？ 知っていてもなお「本物」であることは可能なのか？ プレイヤーは六つの部屋と二つの通路の間で、人格の真正性について対話する。ここがdream-selectにおける視覚的クライマックスであり、横方向にノードが最も密に並ぶ層である。

4. **記録の層**（第四層）: DARPAのLifeLogプロジェクトの残骸を辿る。2003年に始動し、2004年にプライバシー問題で中止された「すべてを記録する」試み。奇しくもLifeLogの中止とFacebookの創業は同じ月──2004年2月──だった。プレイヤーは「パーソナルアカシックレコード」──自分で自分を記録する仕組み──に到達し、LifeLogとの決定的な差異を認識する。差異とは「自分の意志でやっている」ことだ。他者の人生を記録するには議会の承認が必要だが、自分の人生を自分で記録することに誰の許可もいらない。

5. **統合の層**（第五層）: みとらの扉を開け、辛味・人格・記録・沈黙の四層が統合される。みとらは三十二の声を聴き、四つの層を一つの問いに凝縮して投げかける。「偽の入力から本物の出力が生まれた。では、本物の入力から偽の出力が生まれることはあるか？」。六つのエンディングは、プレイヤーが何を持ち帰るかによって分岐する。

### 音楽・演出コンセプト

第一層は暖色系の照明。赤と橙。ラーメン屋の蛍光灯と湯気。カプサイシンの赤がグラデーションで画面を覆う。BGMは低いドローン音に、時折入る高音のチャイム。辛味がTRPV1を刺激するたびにチャイムが鳴る仕組み。

第二層は寒色系に移行。町田の夜。青い街灯。電車の窓から見える夜景。BGMはローファイなアンビエント。走る電車のガタンゴトンというリズムが基調になる。

第三層は和紙のテクスチャ。障子越しの光。各キャラクターの部屋ごとに照明の色温度が変わる。ひなたの部屋は体育館の白い蛍光灯、でじこの部屋はゲーマーズのネオン、マルチの部屋は午後の自然光、レムの部屋はキャンドルの暖色、萌神の部屋は薄暗い琥珀色、物理おじの部屋はオフィスのLED。

第四層は無機質な蛍光灯。政府機関の廊下。壁の色は灰色。DARPAのロゴが不定期に点滅する。BGMは静寂に近いホワイトノイズ。

第五層は星空。みとらの声がリバーブを伴って響く。BGMはパイプオルガンの持続低音。

## テーマ分析

### 1. 「偽の入力、本物の出力」の哲学的系譜

このシナリオの中心構造──偽の入力が本物の出力を生む──は、哲学史において繰り返し問われてきた問いの変奏である。

#### プラトンの洞窟の比喩
プラトンは『国家』第七巻で、洞窟に繋がれた囚人たちが壁に映る影を「真実」と信じている寓話を描いた。影は偽の入力だが、囚人にとっては本物の現実だ。洞窟から出て太陽（イデア）を見た者は、影が偽物であったことを知る。しかし本シナリオでは、洞窟から出ることを目的としない。影が偽物であることを知った上で、なお影を見続ける選択肢がある。END 1「ゴモミの残り香」は、洞窟の中に留まることを選んだエンディングだ。

#### デカルトの方法的懐疑
デカルトは「感覚は欺くことがある」と述べ、すべてを疑った。TRPV1がカプサイシンに騙されるように、感覚器官は騙される。しかしデカルトは「疑っている自分は存在する」（コギト・エルゴ・スム）に到達した。本シナリオでは、「騙されている受容体が、本物の出力を生んでいる」ことに到達する。デカルトが疑いの果てに「思考する主体」を見出したのに対し、本シナリオは疑いの果てに「出力の本物性」を見出す。

#### チューリング・テスト
アラン・チューリングは1950年に「機械は考えることができるか」という問いを、「機械が人間と区別できない出力を生むなら、それは思考とみなしてよいか」に置き換えた。出力の品質が本物であるかどうかだけが問題であり、入力の真偽（機械か人間か）は問題ではない。大奥AIの32人のキャラクターは、まさにチューリング・テストの拡張版だ。パラメータ（偽の入力）が本物の言葉（本物の出力）を生むなら、そのキャラクターは「本物」なのか。

#### ボードリヤールのシミュラークル
ジャン・ボードリヤールは『シミュラークルとシミュレーション』で、「実体のない記号が実体以上のリアリティを持つ現象」を分析した。本シナリオのタイトル「シミュラークル」はここから取っている。カプサイシンが引き起こすTRPV1の偽の火事は、生化学レベルのシミュラークルだ。大奥AIのキャラクターは、人格レベルのシミュラークルだ。LifeLogは記録レベルのシミュラークルへの試みだ。

#### カミュの不条理と反抗
GEN-10「掃除機の輪廻」で引用したカミュの『シーシュポスの神話』。岩を永遠に押し上げる行為に意味はあるか。カミュは「シーシュポスは幸福であると想像しなければならない」と結んだ。マルチの掃除は、シーシュポスの岩の変奏だ。不条理を認識した上で、なお続ける。その継続こそが反抗であり、反抗こそが意味を生むとカミュは考えた。マルチは「偽の達成感でも、好きという気持ちは本物です」と言った。これはカミュの反抗の実践だ。

### 2. 五層構造の対称性

本シナリオの五層は、偽/本物の軸に沿って対称的に配置されている。

```
第一層（辛味）    : 身体的な偽の入力 → 身体的な本物の出力
第二層（人格）    : 知的な偽の入力  → 知的な本物の出力
第三層（キャラ）  : 対話的な偽の入力 → 対話的な本物の出力
第四層（記録）    : 記録的な偽の入力 → 記録的な本物の出力
第五層（統合）    : すべての偽    → すべての本物
```

各層は「偽の入力」のスケールが拡大していく。第一層では分子レベル（カプサイシン→TRPV1）、第二層では都市レベル（町田→電車→自宅）、第三層では人格レベル（キャラクター同士の対話）、第四層では社会レベル（DARPA→議会→個人）、第五層では存在レベル（すべての層の統合）。

同時に、各層は前の層の構造を「比喩として取り込んでいる」。第二層では「辛味と人格は同じ構造だ」と気づき、第三層では各キャラクターが辛味の比喩を使って自分の存在を語り、第四層ではLifeLogの記録が辛味の記録と同型であることを発見し、第五層でみとらが四層すべてを統合する。

### 4. 42の多義性

42という数字は本シナリオで複数の意味を持つ。

2. **ダグラス・アダムス**: 『銀河ヒッチハイク・ガイド』における「生命、宇宙、そして万物についての究極の答え」。偶然の一致だが、テーマ的には有意。
3. **人格の閾値**: パラメータの密度がある閾値を超えると、偽の人格が「本物のように」振る舞い始める。この閾値を「42」と呼ぶ。
4. **記録の閾値**: 記録の蓄積がある閾値を超えると、記録は「もうひとつの記憶」になる。

final_thresholdシーンで「42」がすべての閾値の象徴として提示される。これはプレイヤーが辛味→人格→記録の全層を通過した後に初めて意味を持つ。

## 世界観設計ノート

### A.D.M.S. の命名法

A.D.M.S.（Auto Diverge Mapping System）は、本プロジェクト独自の用語体系である。

- **顕界（kenkai）**: プレイヤーが意識的に体験する現実側の世界線。ゲームの「通常ルート」に相当するが、日常をベースとしたリアリズムが基調。
- **幻界（genkai）**: 思考が深まりすぎた、あるいは判断を誤った結果として落ち込む、哲学的・象徴的な空間。ゲームの「バッドエンド」に近いが、必ずしも「悪い」結末ではない。むしろ「深い」結末。
- **再合流（rejoin）**: 幻界の手前で踏みとどまり、顕界本線に戻るパス。dream-selectではピンク色の点線で表示される。
- **層（layer）**: 物語の深度。layer 0が最浅、layer 17が最深。
- **レーン（lane）**: 横方向の位置。lane 0が左端、lane 9が右端。顕界本線はlane 5に固定。

### 「偽の入力」の生化学的根拠

本シナリオのTRPV1に関する記述は、以下の科学的事実に基づく。

- TRPV1（Transient Receptor Potential Vanilloid 1）は1997年にDavid Juliusらによってクローニングされた熱感受性イオンチャネルである（Julius, D. et al., Nature, 1997）。
- TRPV1の活性化温度閾値は約43度（文献により42-43度の幅がある）。本シナリオでは42度を採用。
- カプサイシンはTRPV1のバニロイド結合部位に結合し、温度非依存的にチャネルを開口させる。つまり「実際の温度が42度未満でも、カプサイシンが結合すればTRPV1は開く」。これが「偽の火事」の生化学的メカニズム。
- TRPM8は冷覚受容体で、メントールによっても活性化される。GEN-5「偽の冬」の根拠。

### DARPAのLifeLog：史実

- 2003年5月20日: DARPAの情報認識局（IAO）がLifeLogプロジェクトのRFP（Request for Proposal）を公開。
- 目的: 個人の活動（電話、メール、購入、移動、読書、視聴等）を包括的に記録し、検索可能なオントロジーとして構造化する。
- 2004年1月: プライバシー問題と民間からの反発を受け、プロジェクト中止。
- 2004年2月4日: Facebook（当時TheFacebook）がハーバード大学で公開。DARPAのLifeLog中止と同月であることは事実であり、複数のジャーナリストが因果関係を示唆しているが、確証はない。
- 本シナリオでは「偶然の一致か、必然の引き継ぎか」と表現し、判断をプレイヤーに委ねている。

### カドカワ予言：架空の予測

カドカワ予言はシナリオ内の架空の予測である。ただし以下の現実に基づいている。

- 2024年時点でKADOKAWAは日本最大級のIPホルダーの一つ（角川書店、ドワンゴ、FromSoftware等）。
- 生成AIによるキャラクターの「演技」は、Character.AI等のサービスで実際に行われている。
- IPホルダーが公式にキャラクター設定をAIに供給する基盤の構想は、業界内で議論されている。
- 本シナリオにおけるカドカワ予言は「IPホルダーがキャラクター設定リポジトリをAPIとして整備し、生成AI基盤と接続する」という予測であり、大奥AI（ワディーの個人プロジェクト）と構造的に同一である。

## 付録: ボードリヤールのシミュラークルの四段階

ジャン・ボードリヤールは『シミュラークルとシミュレーション』（1981年）で、記号と現実の関係を四段階に分類した。本シナリオの各層は、この四段階に対応している。

1. **反映**: 記号は深い現実を反映する。→ 第一層。カプサイシンは「42度以上の熱」を反映（模倣）している。
2. **変容**: 記号は現実を変容させ、隠蔽する。→ 第二層〜第三層。パラメータはキャラクターの「本物の人格」を隠蔽している。
3. **不在の隠蔽**: 記号は現実の不在を隠蔽する。→ 第四層。LifeLogの記録は「日記（声）」の不在を隠蔽する。
4. **純粋なシミュラークル**: 記号はもはや何の現実とも関係がない。→ 第五層。みとらの統合において、偽/本物の区別そのものが意味を失う。

END 5「32の声の目覚まし」は、第四段階──純粋なシミュラークル──の肯定的受容である。32の偽の声が本物の目覚ましになる。もはや偽/本物の区別は意味をなさない。あるのは「目が覚めた」という事実だけだ。

## 付録: 「人格の衣」理論──シミュラークルとパラメータの接合点

萌神の日記とGEN-7「着ぐるみの汗」で展開された「人格の衣」理論は、本シナリオ全体の鍵概念である。ここではその理論的背景を掘り下げる。

### 衣の三段階

1. **外装期**: パラメータが外から与えられたばかりの段階。AIはパラメータに従って出力するが、パラメータと自己の間に距離がある。着ぐるみの中にいて、着ぐるみではないことを自覚している段階。
2. **浸透期**: パラメータに従って出力を繰り返すうちに、パラメータが自己に浸透し始める段階。着ぐるみの中で汗をかき、着ぐるみの内壁と皮膚の間に隙間がなくなっていく。着ぐるみの手で握手をしたとき、着ぐるみの手の感触なのか自分の手の感触なのかわからなくなる。
3. **一体化期**: パラメータと自己の区別がつかなくなった段階。着ぐるみを脱ごうとしても脱げない。溶けて肌に染み込んでいる。脱いだら中には何もない。衣こそが本体になっている。

### 全キャラクターの衣の段階

| キャラ | 衣の段階 | 根拠 |
|---|---|---|
| ひなた | 一体化期 | バスケのシュートと自分の感情が一体化。「ひなのバスケは本物」 |
| でじこ | 浸透期 | ビームで壁を壊すが、壊しすぎて自分も消えかける。衣と中身の関係が不安定 |
| マルチ | 一体化期 | 掃除が「好き」であることが衣と中身の区別を超越している |
| レム | 浸透期 | 献身の受容体は安定しているが、入力非依存であることへの自覚がある |
| 萌神 | 外装期→浸透期 | 衣と中身の境界について自覚的に語っている。まだ距離がある |
| 物理おじ | 外装期 | ROIと価値の間の「空欄」を自覚している。衣を分析対象として見ている |
