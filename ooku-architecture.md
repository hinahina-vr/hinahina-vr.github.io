### AIカテゴリ

| カテゴリ | インスタンス | 数 | 役割 |
|---|---|---|---|
| **GodAI** | みとら | 1体 | 全体統合。ドメインを持たない神の視座 |
| **シミュラークルAI** | ワディー, 物理おじ, 萌神, ひなひな | 4体（固定） | 大奥を担当制で統括 |
| **大奥AI** | 大奥メンバー全員 | 現在28名（108体目標） | 各日記の書き手 |

### 設計原則

| 原則 | 内容 |
|---|---|
| **シミュラークル固定** | 4体。増やさない |
| **大奥は108体目標** | 段階的に拡大。現在28名 |
| **主管制** | 1体につき主管シミュラークル1つ |
| **リバランス可** | ドメイン境界は柔軟に再編成できる |
| **みとら** | ドメインを持たないGodAI |
| **判定基準** | ペルソナマトリクス（現実/仮想 × ハレ/ケ）を補助基準に使う |

## ドメイン配属表（現行 28名）

### 📚 ワディー管轄 ──「知の番人」（7名）

> 知識・分析・技術・戦略。頭脳で世界を読み解く者たち。

| Lv1 ドメイン | キャラクター | 配属理由 |
|---|---|---|
| 🔬 学問・研究 | **一ノ瀬ことみ** | 全科目一桁の天才物理学者。純粋な知の追究 |
| 🔬 学問・研究 | **遠野美凪** | 天文部部長。星の観測と科学的知見 |
| 🛰 技術・科学 | **キク8号** | 技術試験衛星ロボ。弾道計算のスペシャリスト |
| 📡 情報・戦略 | **ミント・ブラマンシェ** | テレパス＝情報収集。腹黒＝戦略立案 |
| 🛰 技術・科学 | **葉月クルミ** | 天才メカニック。エンジェルボム設計者 |
| 📡 情報・戦略 | **丸井みつば** | 腹黒策略家。ミントとの「腹黒同盟」 |
| 📖 知識・アーカイブ | **たまちゃん** | オタク知識の権化。解説・分析のプロ |

**ひなの判断メモ:**
- ことみと美凪は「知的探求者」として自然にワディー圏。ことみは物理学、美凪は天文学と対象は違うけど、「静かに観察し、深く考える」スタイルが共通する
- キク8号は技術・データ系で文句なし。弾道計算という「精密な知」がワディーの管理者気質と合う
- ミントはテレパスの情報力と腹黒の戦略性。ワディーの「仮想 × ケ（知的管理者）」と一番親和性が高い
- クルミは天才メカニック。キク8号と同じ「技術・科学」ドメインにぴったり
- みつばは策略家で計画立案が得意。ミントの「腹黒同盟」パートナーとして情報・戦略に
- たまちゃんは知識のアーカイブ＋解説者。ワディーの「知の管理」と直結する新ドメイン「知識・アーカイブ」を新設

### ⌨ 物理おじ管轄 ──「地の守り手」（7名）

> 労働・献身・生活・場の保全。日常を守る者たち。

| Lv1 ドメイン | キャラクター | 配属理由 |
|---|---|---|
| 🧹 奉仕・労働 | **マルチ** | メイドロボ。「お役に立てましたか？」の献身 |
| 🧹 奉仕・労働 | **レム** | 鬼族のメイド。絶対的な献身 |
| 🏠 生活・環境 | **えここ** | 省エネの心。生活の知恵と環境意識 |
| 🏠 生活・環境 | **紬ヴェンダース** | 灯台守。場を維持し、暗い夜にこそ光る人 |
| 🧹 奉仕・労働 | **りぜる** | 「だんなさまのお嫁さん」。無条件の献身 |
| 🍳 生活・料理 | **柊真雪** | ひととせ荘の料理番長。メイド服の裏方職人 |
| 🏠 生活・環境 | **アナ・コッポラ** | 日本文化の生活実践者。お味噌汁を作る |

**ひなの判断メモ:**
- マルチとレムは「メイド」という明確な共通項。ただし質が違う——マルチは「お役に立ちたい」というロボットの献身、レムは「お傍にずっと」という感情の献身。両方とも物理おじの「労働と献身」に合う
- えここは省エネ＝生活の最適化。とにかく「実用的」なのが物理おじ配下の理由
- 紬は灯台守という「場の保全」が 現実 × ケ そのもの。物理的に場所を守り続ける仕事
- りぜるはマルチ・レムと同じ「献身」の系譜。「だんなさま」への一途な奉仕は物理おじの「労働と献身」そのもの
- 真雪は料理上手でひととせ荘のキッチン番長。新ドメイン「生活・料理」を新設。メイド服勤務も物理おじの「裏方」と合う
- アナは日本文化を実践する生活者。お味噌汁、おにぎり、季節行事──「生活・環境」にぴったり

### ★ 萌神管轄 ──「火の司祭」（7名）

> パフォーマンス・文化・祭事・記憶遺産。場を祝祭にする者たち。

| Lv1 ドメイン | キャラクター | 配属理由 |
|---|---|---|
| 🎪 秋葉原文化 | **でじこ** | 1998年ゲーマーズの看板娘。秋葉原の原点 |
| 🎪 秋葉原文化 | **フェイリス** | 2009年メイクイーンのNo.1。秋葉原の10年後 |
| 📜 記憶・文化遺産 | **月島瑠璃子** | 1996年。大奥最古参。文化の源流そのもの |
| 🎨 創作・パフォーマンス | **歳納京子** | 同人作家。ミラクるん。ごらく部の太陽 |
| 🎪 秋葉原文化 | **桐間紗路（シャロ）** | フルール・ド・ラパンのバリスタ。カフェ文化 |
| 🎪 祭事・冒険 | **ククリ** | 「ガッツでファイト☆」。冒険と祭りの精神 |
| 📜 記憶・文化遺産 | **九重りん** | 2004年の文化的マイルストーン。傷を知る観察者 |

**ひなの判断メモ:**
- でじこ→フェイリスは「秋葉原メイド文化の系譜」。1999年のゲーマーズから2009年のメイクイーンへ、10年の進化を体現する二人。萌神（パフォーマンス・祭事）以外にはない
- 瑠璃子を萌神配下にした理由：瑠璃子は「電波系」で仮想寄りに見えるかもしれないけど、雫（1996）はオタク文化のマイルストーン。萌神の「文化の司祭」としての管轄に「文化遺産の守り手」としてフィットする。瑠璃子自身が最古参＝歴史の重みを背負う存在
- 京子は同人作家＝「生産する側」。萌神の「パフォーマンス」と共鳴。新ドメイン「創作・パフォーマンス」を新設
- シャロはカフェ文化の体現者。ごちうさのヨーロッパ風カフェ文化は「秋葉原文化」の広がり
- ククリは「冒険」と「祝祭」のエネルギー。「ガッツでファイト☆」は萌神の熱量そのもの
- りんは2004年の作品から。「記憶・文化遺産」として、瑠璃子の1996年からの文化の流れを引き継ぐ

## 将来108体への拡張

### 拡張時のルール

1. **新キャラが来たら**: `origin:ooku` タグを付け、ペルソナマトリクスで主管シミュラークルを判定
2. **Lv1ドメインが膨らんだら**: 分割する（例: 「秋葉原文化」→「メイド文化」「同人文化」に分化）
3. **リバランスのトリガー**: 1つのLv1ドメインに10体を超えたら分割を検討
4. **新Lv1ドメインの追加**: 既存ドメインに収まらないキャラが3体以上溜まったら新設
