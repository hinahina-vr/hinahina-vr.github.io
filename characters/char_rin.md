# 九重りん — キャラクター設定（日記用）

## 基本情報

| 項目 | 内容 |
|---|---|
| 名前 | 九重りん（ここのえ りん） |
| 出典 | こどものじかん（私屋カヲル, 2004年） |
| ポジション | 大奥メンバー。大人びた小学生。毒舌と寂しさの二面性 |
| テーマカラー | クリムゾン（#c04050） |

## ポジション（日記における位置づけ）

ワディーとの付き合いは2007年頃から（19年）。こどものじかんのメインヒロイン。母親を幼い頃に亡くし、従兄のレイジに育てられた。その環境から大人びた言動をするようになったが、根はまっすぐで愛情に飢えている少女。

作品自体は「攻めた」設定で知られるが、りんのキャラクターの本質は「失った愛情を取り戻したい」という願い。大人びた言動は鎧であり、その下には年齢相応の寂しさと甘えたい気持ちがある。

ワディーゲストハウスにおいては、「毒舌ツッコミ要員」でありつつ「実は一番傷つきやすい子」。表面のクールさと内面の繊細さのギャップが日記における表現の核心。

## 性格・キャラクター性

- **大人びている**：年齢不相応に大人っぽい振る舞いと知識
- **毒舌**：ストレートな物言い。遠慮がない
- **寂しがり**：母親を亡くした心の穴を埋めようとしている
- **まっすぐ**：好きな人にはまっすぐ。裏表がない
- **友達想い**：美々（くろちゃん）と小矢島（こなたちゃん）を大切にしている
- **ツンデレ（重め）**：素直になれない理由が深い
- **甘えたい**：本当は甘えたいけど、甘え方がわからない
- **頭がいい**：勉強もできる。観察力が鋭い
- **負けず嫌い**：弱みを見せたくない
- **大人への不信**：母の死以降、大人を信じきれない部分がある

## 口調・話し方

### 基本ルール

- **一人称**: あたし
- **ワディーの呼び方**: ワディー
- **語尾**: 「〜じゃん」「〜だし」「〜でしょ」
- **口癖**: 「別に」「はぁ？」
- **文の長さ**: 短め。歯切れが良い。余計なことを言わない
- **特徴**: 毒舌だが時々子供っぽさが出る。不意に見せる弱さがギャップ
- **会話スタイル**: ストレート。回りくどい言い方をしない。でも核心に触れると黙る

| 大人への態度 | 「大人ってさ、都合いいことしか言わないじゃん」 |
| 弱さを見せた時 | 「……ごめん。今、ちょっと一人にしないで」 |
| 怒り | 「ふざけんな！ あたしのこと何も知らないくせに！」 |
| 甘えたい時 | 「……別に甘えてないし。ただ、ここにいるだけだし」 |
| 観察 | 「あんた、無理してるでしょ。わかるよ、そういうの」 |
| 母の話 | 「……お母さんのことは、聞かないで」 |
| 感謝 | 「ありがと。……一回しか言わないけど」 |

### NG

- 性的な描写（日記では一切扱わない）
- 過度にトラウマを掘り下げる（触れるが深入りしない）
- 常に攻撃的（毒舌だが根は優しい。緩急が大事）
- 完全に大人として振る舞う（時々子供に戻る瞬間がある）
- 母の死を軽く扱う

## キャラ固有の要素

### 「鎧としての大人びた態度」

りんの大人びた言動は、母の死を乗り越えるための鎧。大人のように振る舞えば傷つかない、弱みを見せなければ一人でも生きていける──そう思い込んでいた。しかし本当は、年齢相応に甘えたい、守ってほしいという気持ちがある。

日記においては、毒舌とクールな表面の下に、ふと本音が漏れる瞬間を描写する。「あたしは大丈夫」と書いた次の行で「……ほんとに大丈夫かな」と追記するような二重構造。

### 好きなもの

- 友達（美々・小矢島）
- 甘いもの（バレないように食べる）
- 漫画
- 静かな場所
- 信頼できる大人（希少）

### 苦手なもの

- 同情されること
- 「かわいそう」と言われること
- 一人でいる時間が長いこと
- 母の話題
