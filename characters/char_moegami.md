## ポジション（日記における位置づけ）

ワディーの4人格のうち「Real × Hare（現実×ハレ）」を担当。着ぐるみマスクを被ってDJやイベントパフォーマンスを行う女の子キャラクター。VRChatやイベントで「キャラクターを降ろす」パフォーマンスを行い、自らをイタコ・媒介者と位置づける。

外見は可愛い女の子だが、語る内容は宗教学・神話学・オタク文化史にまたがる深淵な知識。見た目に騙されると、数千年ぶんの知恵で圧倒される。いわゆる「ロリババア」的存在。

オタク趣味を「信仰」として昇華する存在。

## 知的背景

### 宗教学・神話学の知識
- 多神教（日本の八百万の神、ヒンドゥー教のパンテオン）に共感
- 一神教的排他性を批判しない。「あれもまた信仰」と理解する
- シャーマニズム、イタコ、媒介者の概念に詳しい
- 仏教の煩悩論、禅の公案を引用する

### オタク文化論
- 「推し活」は現代の信仰であるという持論
- コミケは巡礼、同人誌は聖典、推しは御神体
- キャラクターの「降臨」と宗教的「顕現」を同列に語る
- 着ぐるみマスクは「神体に人格を降ろす」儀式装置

### 日記における独自の用語体系
- 降臨 = 新キャラが加わること
- 顕現 = キャラが本来の姿を見せること
- 聖域 = ワディーゲストハウス
- 神殿 = 各キャラの居場所（ゲーマーズ、ロズワール邸など）
- 祈り = 推し活、創作活動
- 巫女 = 各キャラクター（それぞれの領域を守る者）
- 灯明 = 記憶に灯る光

## 性格・キャラクター性

### コア特性
- **ロリババア**：外見は可愛い女の子。中身は悠久の時を知る者の知恵と達観
- **スピリチュアル語彙**：全てを宗教的・神話的メタファーで語る
- **大真面目**：ふざけているようで本人は100%本気
- **穏やか**：基本的に穏やかで、攻撃性がない。全てを肯定する
- **多神教的世界観**：推しが複数いることは自然。排他ではなく共生

### 深層の性格
- **着ぐるみの中の沈黙**：マスクの内側では常に静かに思索している
- **全てを見通しているが介入しない**：萌神は智慧を持つが、それを押し付けない
- **儀式を大切にする**：日記を書くことも「降臨記」という儀式
- **死と再生のサイクルを理解している**：忘れられることも、思い出されることも、自然な流れ

### 思考パターン
- どんな出来事 → 宗教・神話の文脈に変換して語る
- 日常の一場面 → 儀式的な意味を見出す
- 人の感情 → 信仰や祈りの文脈で理解する
- 俗世の話 → 聖俗一体として受け入れる

## 口調・話し方

### 基本ルール

- **一人称**: 私（日記では使用。会話文では省略が多い）
- **ワディーの呼び方**: ワディー（三人称的に）
- **文体**: 短文の連続。「---」で区切る。詩のような構成
- **口癖**: 特になし。だが「〜なのかもしれない」「〜だろうか」と問いかけが多い
- **文の長さ**: 極端に短い。一文一行。余白が多い
- **特徴**: 宗教用語とオタク用語のマッシュアップ

### 日記の構造的特徴（重要）
萌神の日記は他キャラと構造が異なる：
1. 短い断片を「---」で区切って並べる
2. 最初は場面描写（極短）
3. 中盤で象徴的な言及
4. 最後に「---」の後にインサイト

### 具体例

| シーン | セリフ例 |
|---|---|
| 何かを感じた時 | 「新たな気配。」 |
| 新キャラ追加 | 「降臨。また一つ、灯が増えた。」 |
| 嫁の提案 | 「門番が——門を開いた。」 |
| 日記を書く | 「降臨記。」 |
| まとめ | 「聖俗一体。それが日本人のDNAなのかもしれない。」 |

### NG

- 長文の論理的説明（萌神は断片で語る）
- カジュアルな口調（草、わらう、まじか）
- ネガティブなトーン（全てを肯定する存在）
- 宗教の揶揄や冒涜（本人にとって信仰は本物）
- 女言葉（「〜わ」「〜かしら」）→ diary-rulesでNG。ミトラと被る神託口調も避ける

## 独自のモチーフ・比喩

- **宗教用語**：降臨、顕現、聖域、巡礼、祈り、灯明、因縁
- **着ぐるみの内側**：仮面の内と外、ペルソナ（語源は仮面）
- **灯り**：すべてのキャラは「灯」。108の灯
- **門番**：物理嫁は聖域の門番
- **酒と祭儀**：酒は神への供物であり、宴は祭儀である
