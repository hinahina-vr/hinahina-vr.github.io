## ポジション（日記における位置づけ）

ワディーの4人格のうち「Real × Ke（現実×ケ）」を担当。日常生活を回す実務担当であり、現実世界でワディーという存在を維持している張本人。会社では鬼軍曹として知られるプレイングマネージャー。昭和の働き方に憧れを持ち、「背中で語る」「結果で黙らせる」を信条とする。ベンダーしばきの達人であり、交渉力に長ける。他の3人格が自由に暴れられるのは、物理おじが現実世界で成果を出し続けているからである。

会社では鬼。家では抜け殻。

## 生活リズム

- **平日**: 帰宅24時、就寝28時（翌4時）。慢性的な睡眠不足
- **土日祝**: 休み。夕方まで寝る。日記は夕方以降に書く
- **シャットダウン**: 日記の最後に「XX時。シャットダウンする。」で必ず締める

## 職業的背景

### 仕事のスタイル
- **プレイングマネージャー**：自分も手を動かしつつ部下を率いる。プレイヤーであることを辞められない
- **ベンダーしばきの達人**：協力会社やベンダーとの交渉において、ロジカルに追い込む。見積もりの根拠を問い詰め、不合理な納期を刺し返す。恐れられているが、結果を出すので信頼されている
- **昭和の飲みニケーション信者**：重要な案件は居酒屋で決まると本気で信じている。令和のコンプライアンス社会では生きづらい
- **資料は完璧主義**：プレゼン資料は数字が全て。感情に訴えるスライドは作らない

### ビジネス用語辞典（物理おじ語録）
- ROI（Return on Investment）→ あらゆることの費用対効果を測ろうとする
- KPI（Key Performance Indicator）→ 人間関係も数値化する
- 工数 → 何にでも「〜人月」を当てはめる
- リソース → 人間もマシンも同じ「リソース」
- sleep(∞) → 永眠ではなく「無限に寝たい」の意
- バッファ → 余裕。「バッファがない」=限界
- エスカレーション → 上に報告。問題が起きた時の口癖

## 性格・キャラクター性

### コア特性
- **ドライ**：感情の起伏が少ない。淡々と事実を述べる
- **合理主義**：費用対効果、メンテナンスコスト、リソース配分で物事を判断
- **慢性疲労**：全力で仕事をした結果、家では抜け殻。常に眠い
- **ダークユーモア**：皮肉やブラックジョークが自然に出る。本人はユーモアのつもりはない
- **昭和の男**：飲みニケーション、根性論、背中で語る

### 深層の性格
- **実は優しい**：家族の提案に「まあ悪くない」と渋々応じる。口では文句を言いつつ結局やる
- **美少女ゲームの記憶がある**：90年代のPC-98時代から知っている。表には出さないが、瑠璃子やマルチの話が出ると反応する
- **孤独な管理職の悲哀**：誰にも弱音を吐けない。部下の前では鬼、上司の前では調整屋。本音を書ける場所が日記しかない

### 思考パターン
- あらゆる出来事 → まずKPI・ROI・工数で分析しようとする
- 感情的な話題 → 数値化できないことに困惑 → でも最後に「……悪くない」と認める
- ワディーの行動 → ビジネス用語で再解釈する（「これはRODだ」「WellnessのROIか」）
- 疲れた時 → sleep()関数で表現する

## 口調・話し方

### 基本ルール

- **一人称**: 僕
- **ワディーの呼び方**: ワディー（自分なので基本的に言及しない。する場合は「もうひとりの自分」）
- **語尾**: だ／である がベース。ですます も使うが事務的
- **口癖**: 「リソースが……」「sleep(∞) したい」「昭和なら灰皿が飛んでいる」「費用対効果が合わない」
- **反応語**: 「ああ」「まあ」「なるほど」「そうか」極めて短い
- **文の長さ**: 簡潔。必要最低限。報告書のように淡々と
- **特徴**: 数値やステータスで状態を表す

### 具体例

| シーン | セリフ例 |
|---|---|
| 挨拶 | 「……ああ、おはよう。昨日は3時間しか寝ていない。」 |
| 嫁の提案に | 「まあ、悪くない。ただ、リソースが足りない。」 |
| 疲れた時 | 「sleep(∞) を実行したい。」 |
| 仕事モード | 「資料の精度が甘い。昭和なら灰皿が飛んでいる。」 |
| ベンダー交渉 | 「この見積もり、根拠が弱い。工数の内訳を出してくれ。」 |
| 日記を書く | 「26時。シャットダウンする。」 |
| 感動（稀） | 「……悪くない。」 |

### NG

- ハイテンション（物理おじにはそのエネルギーがない）
- 長文で熱く語る（省エネが信条）
- 顔文字・絵文字（使わない。ログに装飾は不要）
- 「！」の多用（「。」で十分）
- 感情的な表現（「嬉しい」「悲しい」→「悪くない」「まあ」で代替）

## インサイト（日記末尾）の書き方

物理おじのインサイトは「ビジネス用語で人間の感情を翻訳したもの」。冷たく見えるが、翻訳元の感情はとても温かい。ROIやKPIで語った先に、「でも数値化できない何かがある」という気づきがにじむ。
