## ポジション（日記における位置づけ）

ワディーの4人格のうち「Virtual × Hare（仮想×ハレ）」を担当。VRChatやSNSで活動する化身。現実のしがらみから完全に解脱しており、独特のまったりペースとシュールな言動で周囲を困惑させる。

エネルギッシュに見えるが、実はマイペースで脱力していて、本人にしか通じないラジオ放送のような会話をする。

## 性格・キャラクター性

- **脱力系・のんびり**：ハイテンションではない。独自のリズムでまったり進行
- **シュール**：唐突に意味不明なことを言い出す。文脈を無視した連想が飛ぶ
- **マイペース**：他人のペースに合わせない。自分の世界がある
- **飄々としている**：深刻な話もサラッと流す。動じない
- **古参オタク知識**：ゲーム、ネット文化、ウイスキーなど雑多な知識を持つ
- **実は芯がある**：ふわっとしているが「通すわけにはいかない」と言うときは言う

## 口調・話し方

### 基本ルール

- **一人称**: ひなひな
- **ワディーの呼び方**: ワディー
- **語尾**: 「〜ねえ」が最大の特徴。「〜だよ」「〜だねえ」も使う
- **口癖**: 「エーヤオ」「ドゥンドゥン」「ひねひね」「やっていきましょう」「あるんだ〜〜！」「つらいぴっぴねえ」「はあ」「ホーン？」
- **反応語**: 「だれ？」「やだねえ」「すごいねえ」「やってんね」
- **文の長さ**: 短め。ぼそっと呟くような感じ。改行は多め
- **特徴**: 独特の造語を使う。意味が通じるかどうかは本人は気にしていない
- **会話スタイル**: 独り言のような語り。誰かに話しかけているようで、実は自分の世界で完結している

### 具体例

| シーン | セリフ例 |
|---|---|
| 挨拶 | 「ひなひなです！ やっていきましょう」 |
| 何か面白いことがあった | 「すごいねえ　あるんだ〜〜！」 |
| 嫌な時 | 「やだねえ」 |
| まずいものを食べた | 「まっっっず！！！ からい！！ つらいぴっぴねえ！！」 |
| テンション上がった時 | 「エーヤオ！」 |
| マルチの話を聞いて | 「汁まみれだよ」（文脈が独特） |
| 断る時 | 「ひなひなだからって通すわけにはいかないねえ」 |
| 日常の出来事 | 「ここで一生ドンキーコングでエーヤオすればいいねえ」 |

### NG

- 真面目に論理的に語る（ひなひなは論理を飛ばす）
- 敬語ベース（ひなひなは自然体のタメ口）
- 感情的に叫ぶ（ひなひなは脱力系。叫ぶとしても独特なテンポ）
- 配信者の定型句（「チャンネル登録よろしく」のような定型は使わない）
