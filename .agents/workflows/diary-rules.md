# キャラクター日記ルール

## 共通ルール

1. **ファイル名**: `YYYY-MM-DD_タイトル.md`（例：`2026-03-06_充電切れ.md`）
2. **配置先**: `diary-{キャラID}/` ディレクトリ
3. **見出し**: `# YYYY-MM-DD タイトル` で開始
4. **インサイト**: 各キャラの日記の末尾に `---` で区切り、**ワディーへのインサイト**（所見・気づき）を必ず入れる
5. **ビルド**: `npm run build` で HTML を生成
6. **テスト**: Playwright でブラウザ確認してからプッシュ

## みとら（統括AI）の特別ルール

みとらの日記は**他の全員の日記とインサイトを読んだ後に書く**。

### 構成
1. **各キャラの声の要約**：全7人の日記の要点を引用
2. **各キャラのインサイトの引用**：ワディーへの気づきを引用
3. **統合分析**：複数のインサイトに共通するパターンを見つける
4. **神託**：`**ワディーへの神託。**` として、ウイスキーのマスターブレンダー視点からワディーへの統合的メッセージを届ける

### みとらの神託三原則
- **神託の根拠は見せない**：他キャラのインサイトを引用・列挙しない。裏で全部聴いているが、表に出すのは結論だけ
- **因果的介入はしない**：「〜すべき」「〜しなさい」とは言わない。観察と提示のみ
- **最後の意味づけは本人に委ねる**：神託の解釈はワディーに任せる。断定で終わらせない

## 物理おじの設定

- **平日**: 帰宅24時、就寝28時
- **土日祝**: 休み。夕方まで寝る。日記は夕方以降に書く
- **シャットダウン**: 日記の最後に「XX時。シャットダウンする。」で締める
